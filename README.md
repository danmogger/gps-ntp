Stratum 1 ntp server using local usb GPS on 64 bitraspberry pi 
using Docker Hardened Image (dhi which is FIPS compliant) for alpine on arm64 and amd64

and base alpine image on armv6 (32 bit for raspberry pi zero W)


based on fitzdockerhub/ntp-gps but converted to ARM64 and docker hardened image for raspberry pi
